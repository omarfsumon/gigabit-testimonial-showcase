== Description ==

A simple and clean testimonial showcase plugin with custom post type, star rating and shortcode display.

== Installation ==

1. Upload the plugin to `/wp-content/plugins/`
2. Activate through the 'Plugins' menu.

== Usage ==

Use the shortcode to display testimonials:

Basic:
[testimonials]

Custom:
[testimonials count="6" layout="grid" limit_text="150"]

== Frequently Asked Questions ==

= How do I change the number of testimonials? =
Use the `count` attribute.

= What is the custom post type name? =
`testimonial`
