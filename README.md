# Gigabit Testimonial Showcase

**Gigabit Testimonial Showcase** is a lightweight, responsive, and modern WordPress plugin that allows you to showcase client testimonials on your website beautifully. It’s easy to install, customize, and fully developer-friendly.

## Features

- Clean, modern, and fully responsive design.
- Default 3-column grid layout.
- Shortcode ready: `[testimonials]`.
- Dynamic shortcode with multiple options: `[testimonials count="6" layout="grid" limit_text="150"]`.
- Fully customizable colors, fonts, and styling via plugin settings.
- Supports unlimited testimonials.
- Lightweight and fast-loading.
- Developer-friendly with hooks and filters.

## Installation

1. Download the plugin zip file.
2. Go to WordPress Dashboard → Plugins → Add New → Upload Plugin.
3. Upload `gigabit-testimonial-showcase.zip`.
4. Activate the plugin.
5. Go to **Testimonials → Add New** to create your first testimonial.
6. Use the shortcode `[testimonials]` or dynamic shortcode anywhere on your pages, posts, or widgets.

## Usage

### Default Shortcode

```php
[testimonials]
```

### Dynamic Shortcode

```php
[testimonials count="6" layout="grid" limit_text="150"]
```
